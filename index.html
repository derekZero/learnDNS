Domain Name System


DNS covers the relationship between a client (browser) and a client (server) to establish the name of a website. Specifically, cPanel users encounter DNS issues most frequently upon intial setup of their website.

CNAME Record or A Record?

C[anonical] Name Record is used to a map a name from another name. More commonly used is the A[ddress] Record, which maps a name to one or more IP addresses when the IP are known and stable.

What's FTP?

File Transfer Protocol is the most standard network protocol used to move files between a server and a client. It's had some sisters and cousins since then, which brings us too..

Did you just add a letter?

Secure File Transfer Protocol's most identifying feature is it's use of SSH (Secure Shell), a client-server model. It ensures that the data transmission occurs over a secure connection and is verified on the server. It's most useful for managing information on a remote computer. Frequently udpated websites could benefit from SFTP.

Secure Mail

Simple Mail Transfer Protocol has none of the pop or flavor of POP3 or IMAP as it can't receive your mail, but it's enough to relay. Which is probably all you'll need your website to do unless you plan to compete with Google and Yahoo.
